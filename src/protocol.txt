Протокол сетевой игры "Морской бой"
Версия: 1.0
Формат сообщений: текст (ASCII), одно сообщение в строке, завершается символом новой строки '\n'.

Каждое сообщение состоит из команды и, при необходимости, аргументов, разделённых пробелами.
Команды пишутся ЗАГЛАВНЫМИ буквами.

------------------------------------------------------------
ОБЩИЕ КОМАНДЫ
------------------------------------------------------------

HELLO <name>
    Отправляется клиентом серверу при подключении.
    <name> — имя игрока.
    Пример: HELLO Player1

WELCOME <name>
    Отправляется сервером клиенту в ответ на HELLO.
    <name> — имя сервера.
    Пример: WELCOME HostMaster

READY
    Объявление готовности к игре.
    Может быть отправлено обеими сторонами.

START
    Отправляется сервером клиенту.
    Сигнал о начале игры. После этой команды начинается первый ход.

BYE
    Завершение соединения. Обе стороны могут отправить.

------------------------------------------------------------
ИГРОВЫЕ КОМАНДЫ
------------------------------------------------------------

SHOT <cell>
    Выполняет выстрел по указанной клетке.
    Отправляется текущим игроком противнику.
    <cell> — координата поля, например A1, B5, J10.
    Пример: SHOT C3

RESULT <cell> <status>
    Ответ на выстрел.
    <cell> — координата выстрела.
    <status> — результат: MISS, HIT, DEAD.
    Примеры:
        RESULT C3 MISS
        RESULT E7 HIT
        RESULT A1 DEAD

TURN
    Передача хода.

RESTART
    Предложение начать игру заново.
    Может быть отправлено любой стороной.

ACCEPT
    Подтверждение на предложение RESTART.

DECLINE
    Отказ от предложения RESTART.

------------------------------------------------------------
ДОПОЛНИТЕЛЬНЫЕ КОМАНДЫ
------------------------------------------------------------

PING
    Проверка активности соединения.

PONG
    Ответ на команду PING.

------------------------------------------------------------
ПРИМЕР СЕАНСА
------------------------------------------------------------

Client → Server: HELLO Player1
Server → Client: WELCOME HostMaster
Client → Server: READY
Server → Client: READY
Server → Client: START

Client → Server: SHOT C3
Server → Client: RESULT C3 MISS
Server → Client: TURN

Server → Client: SHOT E7
Client → Server: RESULT E7 HIT
Server → Client: TURN

Client → Server: SHOT A1
Server → Client: RESULT A1 DEAD
Server → Client: WINNER HostMaster
Server → Client: BYE
